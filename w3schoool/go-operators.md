### Go Operators
The go operators are used to perform many different operations like take the + for instance, it takes to values and add them together 
Exampole:
```go
package main
import ("fmt")

func main() {
  var a = 15 + 25
  fmt.Println(a)
}
```

Type of operators are as follows:
* Arithmetic Operators: Arithmetic operators are used to perform our mathematical operations. e.g Addition, Subtraction, Multiplication, Division and so more

* Assignment Operators: as the names says it is used to assign values to a variable and also assign values to a variable mathimaticaly like =, +=, -= etc

* Go Comparison Operators: just as the name sound like it is use dto compare two variables the return value of a comparison is either true (1) or false (0). Examples of an comparison operatorn are ==, != >, <, >, etc

* Go Logical Operators: These operators are used to make the logic between variables or values examples are &&, ||, !.

* Go Bitwise Operators: Bitwise operators are used on (binary) numbers examples are &, |, <<, >>, etc.
