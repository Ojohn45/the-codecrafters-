### Go Slices
slice are very much similar to array similar to array but more flexzible and easier to use than array and even more efficeint than it.
Like arrays, slices store multiple values of the same type in a single variable. But this time the lenght of a slice can grow and shrick as you want it to be. In Go, there are many ways to create a slice:
* Using the []datatype{values} format
* Create a slice from an array
* Using the make() function
