### Go Conditions
Conditional statements are used to do different actions based on different conditions you give. 
These conditions in go can be either true or false. Some example of coditional statment are:
* Less than <
* Less than or equal <=
* Greater than >
* Greater than or equal >=
* Equal to ==
* Not equal to !=

You can use if statement to specify a block of code to be executed in a program or use else statement to specify a block of code to executed, if the same condition is false, or use else if to specify a new condition to test, if the first condition is false, or use switch to specify many alternative blocks of code to be executed
