### Go For Loops
Here’s a simple and easy-to-understand summary:

A for loop in Go is used to repeat a block of code multiple times. It is the only loop available in Go, and each time it runs is called an iteration.

The loop usually has three parts:

Start value (where counting begins)
Condition (when to stop)
Increment (how the value increases each time)

For example, it can count numbers like 0 to 4 or even 0 to 100 in steps of 10.

There are also some useful controls:

continue → skips the current loop step and moves to the next one
break → stops the loop completely

You can also use nested loops, which means putting one loop inside another. This is useful when working with combinations, like matching items from two lists.

Finally, Go has a range keyword, which makes looping through arrays, slices, or maps easier. It gives you both the index (position) and the value of each item.

In short:
A for loop helps you repeat code, control how it runs, and work easily with collections of data.
