### Go Output Functions
Array is used to store multiple values of the same data type in one varaiable, so instead of declaring differente variables for every data tpye you come across you use array to groop them.

### Declaring an Array
if you want to declare a varaible in array you canused the var keyword.
Example:

* var array_name = [length]datatype{values} // here length is defined


* var array_name = [...]datatype{values} // here length is inferred 

Or you use the := sign:
Example:
* array_name := [length]datatype{values} // here length is defined

* array_name := [...]datatype{values} // here length is inferred 

You canalso use array for many activities like Change Elements of an Array, Array Initialization, Initialize Only Specific Elements, Find the Length of an Array,
